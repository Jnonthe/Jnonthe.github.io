Dreamweaver Tutorial

Dreamweaver Tutorial (Step-2)

(How to make a basic layout)

By: Joel Nonthe

Welcome to step 2! You made it this far awesome. Once you clicked create your program should look like this. Dreamweaver helps by including this template. Make sure to include all code and insertion points inside of the body. We will start by making a div which is division a section. Start by clicking the insert tab at the left top corner. A drop down menu should appear and click div. Next this screen should appear.
Next this window will show up. Make sure your insertion point is in the body and keep the at insertion point option. click new css rule.
After clicking new css rule you will give your div a name and make it a class or id. This window should show up. This next step is important!
for now we will use id's and not classes because we are making only one page. In the selector type category click the arrow and select id, its should give you a description of what it does. Next choose an appropriate name for your new div id.
Once name is made click ok.
Once clicked this window should pop up.
Once this window pops up click block in the category panel.
The window should change to this and you can start by making the header.
Make sure to always set width to 100% for now this is what we will do. You can change it to your disired width but for this tutorial we will use 100% set a height of 40px and in background choose a light gray background then click ok.
This is what your live view should look like you can start seeing the header of a website. of course it can be arranged in many ways but for now we will make it together.
